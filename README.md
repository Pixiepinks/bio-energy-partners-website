# Bio Energy Partners Website

Production-ready corporate website foundation for Bio Energy Partners (Pvt) Ltd.

## Core message

> We Manage Your Boiler Using Sustainable Bio Energy.

## Technology stack

- React 19
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Lucide React Icons

## Project structure

```text
src/
  assets/
  components/
  layouts/
  pages/
  sections/
  hooks/
  utils/
  styles/
  data/
```

## Available scripts

- `npm run dev` - start the Vite development server.
- `npm run build` - create a production build.
- `npm run preview` - preview the production build locally.
